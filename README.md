## Local Development

Want to run this splashy project on your own machine? It is super easy!

1. Clone the repository to your local computer.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Fire up the Vite development server:
   ```bash
   npm run dev
   ```
